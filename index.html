The Slidorian

A combination of an image slider and an accordian, the Slidorian displays beautiful images along with a variable length description. With images linked to each tab, and accompanied by a large array of effects, the slidorian is a great alternative to the traditional jQuery slider.

Download

Usage

Import jQuery, jQuery UI and Slidorian

<link rel="stylesheet" href="css/slidorian.css" /> <link rel="stylesheet" href="css/jquery-ui.css" /> <script src="js/jquery.min.js"></script> <script src="js/jquery-ui.min.js"></script> <script src="js/jquery.easing.js"></script> <script src="js/slidorian.js"></script>

HTML Structure

<div id="slidorian"> <div id="slider"> <div class="slider-image" rel="section1"> <img src="sample.jpg" width="500" height="400" /> </div> <div class="slider-image" rel="section2"> <img src="sample.jpg" width="500" height="400" /> </div> <div class="slider-image" rel="section3"> <img src="sample.jpg" width="500" height="400" /> </div> </div> <div id="accordian"> <div class="header"><a class="slider-link selected" rel="section1">Kung Fu Panda 2</a></div> <div class="content"> <!-- CONTENT GOES HERE --> </div> <div class="header"><a class="slider-link" rel="section2">The Dark Knight</a></div> <div class="content"> <!-- CONTENT GOES HERE --> </div> <div class="header"><a class="slider-link" rel="section3">Hot Fuzz</a></div> <div class="content"> <!-- CONTENT GOES HERE --> </div> </div> </div>

Basic Initialising

<script> $(document).ready(function(){ $('#slidorian').slidorian(); }); </script>

Initialising With Options

<script> $(document).ready(function(){ $('#slidorian').slidorian({ speed: 1000, interval: 4000, effect: 'slideLeft' }); }); </script>

Options

  • autoPlay

    Automatically plays the slidorian (boolean)

  • easing

    The easing of the animations (string)

  • effect

    The effect of the animation (string)

  • first

    The first slide to show (the rel of that slide) (string)

  • hoverPause

    The slidorian pauses when hovered over (boolean)

  • interval

    The time between each slide (number)

  • speed

    The speed of the animation (number)
